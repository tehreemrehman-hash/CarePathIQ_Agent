## Key Takeaways

### The Problem (One Sentence)
The app tried to apply UI design principles (H1, H3, H6-H10) directly to pathway data, which is impossible.

### The Solution (One Sentence)
Split heuristics into 3 actionable (H2, H4, H5) and 7 design-only (H1, H3, H6-H10), apply actionable ones collectively.

### The Safety (One Sentence)
Three focused code changes, backward compatible, easy to rollback.
